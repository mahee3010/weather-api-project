# Weather API Project

## Overview
This project fetches weather forecast data using the RapidAPI Weather API, cleans the JSON response, and converts it into a structured dataset suitable for analysis.

## Tools Used
- Python
- Jupyter Notebook
- Pandas
- Requests
- RapidAPI

## Workflow
1. Fetch weather data using API
2. Normalize JSON response into a DataFrame
3. Clean unnecessary fields and columns
4. Save the clean dataset as CSV

## Notes
- API keys are not included in this repository for security.
- All temperatures are in Kelvin.
- Cloudiness and other measurements include units in the same column.
